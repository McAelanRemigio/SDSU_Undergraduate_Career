## MATH 254 – Introduction to Linear Algebra

### Course Focus
- Foundations of linear algebra for mathematical, statistical, and computational applications
- Emphasis on both theoretical understanding and computational implementation
- Development of linear reasoning for modeling, data analysis, and STEM applications

### Core Topics Learned
- Systems of linear equations and matrix methods:
  - Gaussian elimination
  - Matrix operations, inverses, and determinants
- Vector spaces and subspaces:
  - Linear independence, basis, and dimension
  - Row space, column space, and rank
- Linear transformations:
  - Mappings between vector spaces and their properties
- Eigenvalues and eigenvectors:
  - Computation and applications
- Geometric interpretations:
  - Norms and inner products
  - Orthogonality and projections
  - Gram–Schmidt process

### Mathematical & Computational Skills Developed
- Solving and analyzing linear systems using matrix methods
- Reasoning abstractly about vector spaces and transformations
- Connecting algebraic operations to geometric interpretations
- Using computational tools (e.g., MATLAB) to support analysis and verification
- Applying linear algebra concepts to data-oriented and modeling contexts
