### Course Focus
- Introduction to Python programming with an emphasis on data collection and data analysis
- Development of computational thinking for working with real-world data
- Building foundational programming skills for students with no prior coding experience
- Using Python as a practical tool for data manipulation, visualization, and analysis

### Core Skills & Concepts Learned
- Python programming fundamentals:
  - Python data types and variables
  - Functions, control structures, and program flow
  - Writing clear and reusable code
- Data computation and representation:
  - Working with matrix and array-based data using NumPy
  - Understanding computational representations of data
- Data manipulation and aggregation:
  - Cleaning, transforming, and summarizing data using pandas
  - Grouping, filtering, and aggregating structured datasets
- Data analysis and modeling:
  - Using computational tools to support statistical reasoning
  - Introduction to machine learning concepts through scikit-learn
  - Applying models to larger-scale datasets
- Visualization and interpretation:
  - Creating visualizations to explore and communicate data
  - Using computational tools to support understanding of statistical and structural patterns

### Course Context
This course provides a foundational entry point into Python-based data analysis and computational thinking. It emphasizes practical scripting skills alongside conceptual understanding, preparing students to apply Python tools for data-driven problem-solving in more advanced data science, statistics, and applied analytics coursework.
