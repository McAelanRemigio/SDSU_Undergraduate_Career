### Course Focus
- Introduction to statistical programming and data management using SAS Studio
- Emphasis on data access, cleaning, transformation, and reporting
- Development of reproducible data workflows for structured datasets

### Core Skills & Concepts Learned
- Working in SAS Studio:
  - Developing, modifying, saving, and executing SAS programs
  - Using both point-and-click tools and written SAS code
- Data access and management:
  - Reading data from SAS datasets, CSV, TSV, and Excel files
  - Creating temporary and permanent SAS datasets
  - Subsetting, sorting, and restructuring data
  - Identifying and removing duplicate observations
  - Merging and joining multiple datasets
- Data cleaning and preparation:
  - Detecting and resolving data quality issues
  - Transforming variables for analysis and reporting
- Reporting and output:
  - Producing detail, summary, and statistical reports
  - Generating HTML, PDF, and Excel outputs
- Automation and programming:
  - Writing simple macros
  - Using data-driven programming techniques to streamline workflows
- Data visualization:
  - Creating line, bar, box, and pie charts using SAS statistical graphics
  - Building dashboards that combine visualizations with tabular output
