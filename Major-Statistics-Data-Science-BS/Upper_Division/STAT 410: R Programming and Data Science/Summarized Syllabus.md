### Course Focus
- Applied statistical modeling and data analysis using the R programming language
- Emphasis on regression, ANOVA, and foundational machine learning concepts
- Integration of statistical theory, computation, and communication
- Consideration of ethical and societal implications of data science methods

### Core Skills & Concepts Learned
- Data analysis in R:
  - Importing, cleaning, transforming, and summarizing data
  - Data visualization for exploratory analysis and communication
  - Performing basic statistical inference in R
- Statistical modeling:
  - Linear regression and ANOVA models
  - Interpretation of model assumptions and inference
  - Understanding the mathematical foundations underlying common models
- Reproducible analysis and reporting:
  - Using RStudio for statistical workflows
  - Writing reproducible reports with R Markdown and Quarto
  - Communicating statistical findings clearly to technical and non-technical audiences
- Applied data science:
  - Applying regression-based models to real datasets
  - Exposure to a broad range of machine learning approaches
  - Evaluating model outputs from a data science perspective
- Ethics and societal context:
  - Discussing equity, inclusion, and fairness in data-driven decision-making
  - Examining challenges in deploying machine learning tools in public policy contexts

### Course Context
This course serves as a bridge between traditional statistical methods and modern data science practice. It emphasizes not only technical modeling skills, but also reproducible workflows, communication, and ethical awareness, preparing students for applied data science work in academic, industry, and policy settings.
